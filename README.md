# 📊 Telecom X – Predicción de Churn (Parte 2)

## 📌 Descripción

Este proyecto forma parte del **Challenge Telecom X – Parte 2** del programa **Oracle Next Education / Alura Latam**.
El objetivo fue desarrollar un **modelo predictivo de Machine Learning** capaz de identificar clientes con mayor probabilidad de cancelar los servicios de Telecom X (churn).

Para ello se utilizaron los datos previamente limpiados y transformados en la **Parte 1 del desafío**, aplicando técnicas de preparación de datos, análisis estadístico y modelado predictivo.

---

## ⚙️ Tecnologías utilizadas

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Google Colab

---

## 🔄 Proceso realizado

1. Carga del dataset tratado (`datos_tratados.csv`).
2. Preparación de los datos para Machine Learning.
3. Codificación de variables categóricas.
4. División del dataset en **entrenamiento y prueba**.
5. Entrenamiento de un **modelo predictivo**.
6. Evaluación del modelo utilizando métricas de desempeño.

---

## 📈 Resultados

El modelo permitió identificar variables relevantes asociadas a la cancelación del servicio, como:

* tipo de contrato
* cargos mensuales
* antigüedad del cliente
* servicios contratados

Estos resultados pueden ayudar a la empresa a **anticipar la pérdida de clientes y aplicar estrategias de retención más efectivas**.
